# Arc Raiders Companion App

## Overview
A companion application for the video game **Arc Raiders**. This app is designed to enhance the player experience by providing useful tools, data, and guides.

## Features
- **Data Tracking**: (Planned) Track player stats and inventory.
- **Interactive Maps**: (Planned) Maps for game locations.
